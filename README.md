# Cartify

Cartify is a smart receipt scanner and price tracker app that helps users organize receipts, track spending, search purchased products, and compare prices across stores.

## Features

- Scan paper receipts using your camera
- Extract receipt information automatically using AI/OCR
- Organize receipts and purchases
- Search products you have purchased
- Compare product prices across receipts and stores
- View spending insights and analytics
- Track purchase history over time
- Support for multiple currencies
- Receipt history and category analytics
- Cloud sync with account login

## Technologies

- Flutter
- Flask
- Firebase
- AI/OCR processing
- Apple Sign In
- Google Sign In
